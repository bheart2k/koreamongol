# 1차 개발 회의 자료 조사 (2026-02-14)

## 상세 조사 결과는 사용자에게 직접 전달함
## 이 파일은 핵심 발견사항만 기록

### 발견된 추가 니즈 (MVP에 없는 것)
1. 노동권리 가이드 (임금체불, 산재, 사업장 변경) - Impact 매우 높음
2. 주거 가이드 (Phase 2 계획에 있음)
3. EPS-TOPIK 준비 (Phase 2 계획에 있음)
4. 법적 지원/상담 연결 정보
5. 몽골 식재료/커뮤니티 위치 정보

### 경쟁 환경 핵심
- 몽골어 전용 한국 생활 가이드 사이트 = 없음 (확인된 블루오션)
- 기존 채널은 Facebook 그룹 + 지인 네트워크가 주력
- Kstart는 범용이라 몽골 특화 콘텐츠 부족
- 정부 포털(다누리, 1345)은 존재하지만 UX가 불친절하고 정보가 분산

### 콘텐츠 전략 시사점
- 송금 가이드: 투그릭 직접 환전 중단(2018~) → 송금 전문업체(GME 등)가 핵심
- 환치기 경고: 실제 수요 높으나 불법 → 명확한 경고 + 대안 제시 필요
- 노동 관련: E-9 사업장 변경 3회 제한, 3개월 내 재취업 필수 등 핵심 정보
