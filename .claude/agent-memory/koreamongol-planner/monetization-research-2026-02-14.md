# 수익화 조사 (2026-02-14)

## 구글 애드센스 언어 지원 현황
- 총 44개 언어 지원
- **몽골어: 미지원**
- 한국어: 지원됨
- 근거: https://support.google.com/adsense/answer/9727, https://www.alltop9.com/languages-google-adsense-supports/
- 2020.9.15 정책 변경: 미지원 언어 신규 사이트 수익화 불가

## 우회 방안 (정책적 리스크 있음)
1. 한국어/영어 콘텐츠를 주 언어로 설정 + 몽골어 번역 병기
2. 한국어 콘텐츠 페이지에만 광고 게재, 몽골어 페이지는 광고 미게재
3. 사이트 자체를 이중언어(한국어+몽골어)로 전환

## 소액 후원/기부 시스템 조사 (2026-02-14 추가)

### 추천 순위
1. **toss.me + 카카오페이 송금 링크** (수수료 0%, 타겟 유저 접근성 최고)
2. **Ko-fi** (해외/몽골 본국 유저용, 팁 수수료 0%)
3. Buy Me a Coffee (5% 플랫폼 수수료, Ko-fi 대비 불리)
4. PayPal.me (수수료 높고 타겟 접근성 낮음)

### 핵심 포인트
- 한국 체류 외국인: 토스/카카오페이 사용 가능 (외국인등록증+은행계좌+한국폰)
- 카카오페이 외국인 사용자 260만명+ (2025.12 기준)
- Buy Me a Coffee/Ko-fi: 한국 결제수단(카카오페이/토스) 미지원이 약점
- 토스/카카오페이 송금링크: 공식 임베드 위젯 없음, 링크+버튼으로 직접 구현

## 광고 네트워크 상세 조사 (2026-02-14 추가)

### 국내 - 사용 불가
- **네이버 애드포스트**: 네이버 블로그/포스트/밴드 전용. 외부 사이트 불가. 완전 탈락.
- **데이블**: 일 PV 3,000+ 또는 일 방문자 1,500+ 필요. 초기 단계 불가.

### 국내 - 시도 가능
- **카카오 애드핏**: 외부 웹사이트 가능, 가입 조건 낮음, Next.js 호환. 단 몽골어 사이트 심사 통과 여부 불확실. 수익 매우 낮음.

### 해외 - 추천
- **Adsterra**: 최소 트래픽 없음, 언어 제한 없음, 최소 출금 $5(Paxum)/$25(PayPal). 한국 CPM Tier2 추정 $0.50~$2.
- **PropellerAds**: Adsterra와 유사. $5 최소 출금, 주간 정산.
- **Monetag**: PropellerAds 자매. 동일 조건.
- 공통 주의: 팝언더/푸시 광고 위주 → UX 저하 위험. 배너/네이티브만 선택 사용 권장.

### 현실적 수익 예상 (일 100명 기준)
- 광고: 월 수천 원 ~ $5 (의미 없는 수준)
- 후원: 콘텐츠 충성도 형성 시에만 의미 있음
- 제휴 마케팅(송금업체 CPA 등)이 초기에 가장 현실적

## 수익화 대안 전략 (광고 외)
- 송금 서비스 제휴 (GME, SBI 등) - CPA 수익
- TOPIK 강좌/교재 제휴
- 몽골-한국 항공권 제휴
- 직접 광고 (광희동 몽골 가게, 송금업체 등)
- 후원/기부 모델

## 결론
- 순수 몽골어 사이트로는 애드센스 승인 불가능
- Phase 1 수익화: 토스+카카오페이 후원 링크 + Ko-fi (구현 쉬움)
- Phase 2: Adsterra/PropellerAds 배너 광고 (트래픽 확보 후)
- Phase 3: 데이블, 직접 광고, 제휴 마케팅 (성장 후)
- 한국어 콘텐츠 병행 시 애드센스 가능성 열림
