## 1. 확정된 전략 결정

### 수익 모델
- **애드센스 사용 안 함** (몽골어 미지원이기도 함)
- 수익 축: **후원(Ko-fi) + 자체 서비스 + 대사관 협력 체계**
- 후원은 ko-fi.com/hangulhub 로 연결 (기존 Buy Me a Coffee는 한국에서 결제 불가로 교체, 2026-07-14).
  카카오페이 QR(한글허브와 동일 계정)도 /donate에 병행 배치. 추후 koreamongol 전용 후원 계정 새로 만들 예정
- 장기 유력 서비스 수익 후보: 일자리 섹션(사업주 측 공고 게재 등) — 규제 검토 결과에 따라

### 일자리 서비스 방향
- **직접 매칭(알선)은 하지 않는다.** 형태는 "정보 제공 + 커뮤니티(사업장 후기·구인구직 게시판)"
- 타겟 시장은 정규직이 아니라 **일용직/단기 알바** (유학생 생활비, 당일·익일 일감)
  - 현재 몽골인들은 페이스북 그룹에서 글+댓글 전화번호 교환으로 해결 중
  - 기존 플랫폼(KOWORK, JOBPLOY 등)은 정규직·장기계약용이라 이 시장을 커버 못 함
- 차별화: 검색 가능한 후기·정보 아카이브, 사기 예방, "몽골인 비율(한국인 위주 사업장)" 태그,
  "내 비자로 이 일 해도 되나?" 비자별 가능 여부 안내
- **직업정보제공사업 신고는 지금 하지 않고 추후 신청** (아래 로드맵 참조)

### 로드맵 (순서 확정)
1. **[현재] 정보 제공 단계** — /jobs 허브 심화(알바·일용직에서 자기를 지키는 법: 일당 확인, 체불 대응, 세금, 유학생 허가), 도구 연동. 신고 불필요, 완전히 안전
   - ✅ 2026-07-14 완료: /jobs 알바 허가·일용직 자기 보호 섹션(유학생 시간제취업 규정은 easylaw 2026-06 기준으로 검증·갱신, 구 규정 "주 20시간" 표 수정), 홈 히어로 가치 제안 + 상황별 진입 버튼, 전체 가이드 "마지막 업데이트일" 표기, 사이트 내 검색(Fuse.js, navbar 돋보기 + Ctrl+K)
2. **[다음] 홍보·트래픽 확보** — 몽골인들에게 홍보, 자주 방문하는 사이트로 자리잡기
3. **[그 후] 직업정보제공사업 신고 + 대사관 협업을 동시 추진** — 구인구직 게시판 오픈
4. **[장기] 채용 연계(매칭)** — 개인사업자로는 불가(아래 규제 참조). 법인화 또는 비영리단체 협업 등 구조 재검토 필요

### 도구 연동 (everycalc / 한글허브) — ✅ 1차 완료 (2026-07-14)
housing/jobs/arrival 가이드 링크 + apps 가이드 한글허브 딥링크 반영 완료. 남은 것: 한글허브 몽골 이름 변환 기능 완성 후 딥링크 연동.
- 연동 방식: 가이드 본문에 맥락 링크(LinkCard) + "유용한 도구" 모음 페이지
- **각 링크에 몽골어 설명 필수** (무엇을 하는 도구인지 + 어디에 뭘 입력하는지 한두 줄) — 두 사이트 모두 한국어/영어 UI라서
- everycalc 연동 대상:
  - housing: 평수 변환, 전월세 전환, 복비 계산기
  - jobs: 주휴수당, **일용직 소득세**, 연봉/시급, 4대보험, 퇴직금/실업급여
  - arrival: 영문주소 변환기, 단위 변환기
- 한글허브 연동 대상: 도장 생성기, 폰트 갤러리 (언어 중립이라 바로 가능)
- **한글허브에 "키릴 몽골 이름 → 한글 표기 변환" 기능 추가 예정** (한글허브 프로젝트 쪽에서 작업).
  완성되면 arrival 가이드 등에서 딥링크 연동
- 수요 확인되는 계산기(평수, 주휴수당, 일용직 소득세)는 장기적으로 koreamongol에 몽골어 버전 이식 검토 (환율·퇴직금 계산기 이식 패턴)

## 2. 규제 조사 요약 (2026-07 조사)

### 단계별 신고/등록 요건

| 단계 | 서비스 형태 | 신고/등록 | 난이도 |
|---|---|---|---|
| ① | 순수 정보 제공 (가이드, 권리 안내) | 불필요 | — |
| ② | 커뮤니티 게시판 (이용자가 구인/구직 글 자율 작성) | 직업정보제공사업 **신고** 필요 | 낮음 |
| ③ | 운영자가 공고 수집·게재 | 직업정보제공사업 **신고** 필수 | 낮음 |
| ④ | 직접 매칭·알선 | 직업소개사업 | **높음** |

### 직업정보제공사업 신고 (②③ 단계용)
- 신고처: 관할 지방고용노동관서(고용센터)
- 서류: 신고서 + 수입·지출계획 포함 사업계획서 + 사업자등록증 사본
- **처리 7일, 수수료 무료, 사무실 시설기준 없음, 개인사업자 가능**
- 위반(무신고 영업) 시: 1년 이하 징역 또는 1천만원 이하 벌금
- 판단 기준(고용노동부 행정해석 — 아이돌보미 앱 사례): 운영자가 매칭에 개입하지 않고
  이용자들이 스스로 연락하게 두면, 구인자에게 수수료를 받아도 "직업정보제공사업"(신고제)에 머묾.
  → "본 사이트는 알선하지 않고 정보만 제공한다"를 약관에 명시할 것 (신고제로 남는 법적 근거)

### 직업소개사업 (④ 단계 — 진입 장벽 높음)
- 무료직업소개사업: **비영리법인·공익단체만 가능. 개인사업자 원천 불가**
- 유료직업소개사업: 등록제. 전용면적 10㎡ 이상 사무실 + 보증보험/예치금 1,000만원(개인),
  법인은 납입자본금 5,000만원 이상

### 유학생(D-2/D-4) 시간제취업허가 — 제품 설계에 중대
- 학부 D-2: 학기 중 평일 주 10시간 (석·박사 15시간, 방학·주말 확대)
- D-4 어학연수: 주 20시간, 입국 6개월 경과 + 출석률 90% + TOPIK 2급 등 요건
- **건설업 금지. 제조업은 한국어 4급 이상만 예외 허용. 특수형태근로(배달 등) 금지**
- → 페이스북에 도는 일용직 공고("인천 목재 일" 류) 상당수는 유학생이 하면 **불법 취업**
- 적발 시: 유학생 강제퇴거·출국명령 가능 (2024년 초 적발 212명 중 44명 강제퇴거),
  사업주는 3년간 외국인 고용 제한
- 사이트 대응 장치 (게시판 오픈 시 필수):
  1. "건설·제조 일용직은 유학생 허가 대상 제외, 무허가 취업 시 강제퇴거 위험" 경고 상시 노출
  2. 공고에 업종 태그 → 비자별 가능 여부 안내
  3. E-9은 워크넷/EPS 공식 채널 우선 안내
  4. 약관에 "알선하지 않음" 명시

### 사업장 후기 기능 관련
- 후기는 별점이 아니라 사실 체크리스트(월급 정시 지급, 잔업수당, 기숙사, 한국인-몽골인 비율) 위주
- 실명 비방은 명예훼손·업무방해 소지 → 신고/삭제 프로세스 필수

## 3. ⚠️ 추후 작업 시 추가 조사 필요 (신고 신청 전 필수)

이번 조사는 2026-07 시점 웹 조사 기반이다. **일자리 게시판 오픈/신고 신청 전에 반드시 재확인할 것:**

1. **법령 최신 조문 재확인** — 직업안정법 조문 번호가 출처마다 엇갈렸음. 국가법령정보센터(law.go.kr)에서 직접 확인
2. **유권해석 문의** — "이용자 자율 게시판도 신고 대상인지"는 100% 동일 판례가 없어 유사 행정해석을 유추 적용한 것. 고용노동부 고객센터(1350) 또는 관할 고용센터에 문의
3. **명예훼손 관련 운영 정책** — 후기 서비스 오픈 전 법률 자문 권장
4. **수치·제도 변동 확인** — 유학생 허용 시간, E-9 사업장 변경 규정(권역 제한 도입 방향), 최저임금 등은 매년 바뀜
5. **대사관 협업 구체화** — 대사관은 "매칭 앱"을 상상했을 수 있음. 왜 정보+커뮤니티 형태가 현실적·안전한지(E-9 구조, 규제, 1인 운영) 설명 자료 준비 후 접촉
