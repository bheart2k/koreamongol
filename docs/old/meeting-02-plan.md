## 수익화 전략 옵션

### 옵션 A: 한국어 콘텐츠 추가 → 애드센스 도전
- 한국어는 지원 언어이므로, 한국어 콘텐츠를 추가하면 승인 가능
- 한국인 대상 콘텐츠 예시: "몽골인 고용 가이드", "몽골 문화 이해" 등
- **단점**: i18n 미설계 상태라 구조 변경 필요, 작업량 증가

### 옵션 B: 대안 광고 네트워크 (언어 제한 없음)
| 네트워크 | 최소 트래픽 | 언어 제한 | 비고 |
|----------|-------------|-----------|------|
| Adsterra | 없음 | 없음 | 다양한 광고 형태 |
| PropellerAds | 없음 | 없음 | 팝업/푸시 광고 중심 |
| Infolinks | 중간 | 없음 | 텍스트 내 광고 |

- **단점**: 수익성 낮고, 광고 품질이 떨어질 수 있음

### 옵션 C: 제휴 마케팅
| 제휴 대상 | 수익 모델 | 적합도 |
|-----------|-----------|--------|
| 송금 서비스 (GME, SBI 등) | 가입/송금당 수수료 | 매우 높음 |
| 통신사 (알뜰폰) | 개통당 수수료 | 높음 |
| EPS-TOPIK 강좌 | 수강 등록 | 중간 |
| 광희동/안산 몽골 가게 | 직접 배너 광고 | 신뢰 높음 |

- **장점**: 유저에게도 가치 있고, 콘텐츠와 자연스럽게 연결
- **단점**: 초기 트래픽 없으면 제휴사 관심 끌기 어려움

### 기획이 추천: A + C 병행
1. 단기: 제휴 마케팅으로 첫 수익
2. 중기: 한국어 콘텐츠 추가하여 애드센스 준비
3. 장기: 애드센스 + 제휴 + 직접 광고 병행

## 단계별 로드맵 (안)

### Phase 1: 기반 구축 (1-3개월)
| 우선순위 | 작업 |
|----------|------|
| 1 | 기존 5개 가이드 콘텐츠 보강 (깊이 + 분량) |
| 2 | GA + Search Console 설정 |
| 3 | SNS 채널 개설, 몽골인 커뮤니티에 콘텐츠 공유 |
| 4 | 송금 서비스 제휴 프로그램 조사/가입 |
| 5 | About 페이지 보강 (신뢰성 확보) |
| 6 | FAQ 확충 (20개 이상) |

### Phase 2: 수익화 시작 + 확장 (3-6개월)
| 작업 |
|------|
| 제휴 마케팅 라이브 |
| 대안 광고 네트워크 테스트 |
| 새 가이드 추가 (노동권리, 주거, TOPIK) |
| 한국어 콘텐츠 섹션 추가 (애드센스 준비) |
| 커뮤니티 활성화 |

### Phase 3: 스케일업 (6-12개월, 목표만)
- 애드센스 신청
- 실시간 환율 계산기
- 커뮤니티 UGC 활성화
- PWA / 모바일 앱 검토
- 오프라인 제휴 (광희동/안산)

## 결정 필요 사항

1. **수익화 전략**: A(한국어 추가) / B(대안 광고) / C(제휴) / A+C(병행)?

4. **커뮤니티**: Phase 2까지 미루기? 지금 열기?
