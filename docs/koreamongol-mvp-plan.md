## 1. 프로젝트 개요

### 1.1 서비스명
**KoreaMongol (코리아몽골)**
- 도메인 (예정): koreamongol.com
- 언어: 몽골어 (키릴 문자) 단일 언어

### 1.2 비전
> **"Таны Солонгос амьдралын хөтөч"**
> (당신의 한국 생활 가이드)

한국에 거주하는 몽골인을 위한 올인원 생활 가이드 플랫폼.
초기에는 정보 가이드 중심, 이후 사용자 참여 커뮤니티로 확장.

### 1.3 타겟 사용자
- 한국 체류 몽골인 약 5~6만 명
- E-9 비자 노동자 (이사업, 제조업 등)
- D-2/D-4 유학생 및 어학원생
- 한국행을 준비 중인 몽골 내 예비 이주자

### 1.4 핵심 가치
| 키워드 | 설명 |
|--------|------|
| Нутаг (고향) | 타국에서 만나는 고향 같은 따뜻한 공간 |
| 신뢰 | 정확하고 검증된 정보, 세련된 전달 |
| 실용 | 당장 필요한 정보를 바로 찾을 수 있는 구조 |

## 3. 디자인 시스템

> 상세 비주얼 가이드는 별도 HTML 파일 참조: `koreamongol-design-guide.html`

### 3.1 컨셉
**"Нутаг (고향)" 듀얼 톤**
- 가이드 영역: 세련되고 신뢰감 있는 — "한국 친구가 깔끔하게 정리해서 알려주는"
- 커뮤니티 영역 (Phase 2): 따뜻하고 편안한 — "고향 사람 만난 것처럼 편하게"

### 3.2 컬러

```css
/* 메인 */
--color-navy: #1B2D4F;       /* 신뢰, 한국의 세련됨 */
--color-gold: #D4A843;        /* 소욤보, 따뜻함, 고향 */
--color-terracotta: #C45B3E;  /* 몽골 전통, CTA 버튼 */

/* 배경 */
--bg-sky: #E8F0FE;            /* 가이드 영역 배경 */
--bg-warm: #FAF6F0;           /* 커뮤니티 영역 배경 */
--bg-white: #FFFFFF;          /* 카드, 콘텐츠 */

/* 텍스트 */
--text-dark: #1A1A2E;         /* 본문 */
--text-sub: #6B7280;          /* 부가 설명 */
```

### 3.3 타이포그래피

```css
/* 제목 - Inter (Google Fonts, Cyrillic subset) */
font-family: 'Inter', sans-serif;

/* 본문 - Noto Sans (Cyrillic subset, Ө/Ү 지원) */
font-family: 'Noto Sans', sans-serif;
```

| 용도 | 사이즈 | 웨이트 |
|------|--------|--------|
| 히어로 제목 | 2.5rem (40px) | Bold (700) |
| 섹션 제목 | 1.75rem (28px) | SemiBold (600) |
| 카드 제목 | 1.25rem (20px) | SemiBold (600) |
| 본문 | 1rem (16px) | Regular (400) |
| 캡션 | 0.875rem (14px) | Regular (400) |

### 3.4 로고
텍스트 로고: **Korea**`(#1B2D4F)` + **Mongol**`(#D4A843)`
- 폰트: Inter Bold
- MVP에서는 이것으로 충분, 심볼은 추후 추가

### 3.5 아이콘
- Lucide Icons 사용
- 기본 색상: --color-navy
- 호버/활성: --color-gold
- stroke-width: 2px

### 3.6 레이아웃 원칙
- **모바일 퍼스트** (타겟 유저 대부분 스마트폰 사용)
- 최대 너비: 768px (모바일 최적화), 데스크탑은 max-w-4xl 정도
- 카드 기반 그리드
- 다크모드: MVP에서는 미지원, Phase 2 검토

### 5.2 비자 가이드 (`/visa`)

**제목:** Солонгосын виз бүрэн гарын авлага (한국 비자 완전 가이드)

**섹션 구성:**

#### 섹션 1: 인트로
- 한줄 소개: "한국에 오는 첫 번째 관문, 비자부터 알아보자"
- 이 페이지에서 다루는 내용 요약 (앵커 링크)

#### 섹션 2: 비자 종류별 안내
- **UI: 탭(Tab) 또는 아코디언**으로 전환
- 탭 목록: E-9 고용 | D-2 유학 | D-4 어학 | 기타
- 각 비자별 포함 내용:
  - 대상: 누가 받을 수 있는가
  - 필요 서류 (CheckList 컴포넌트)
  - 신청 절차 (StepList 컴포넌트, 번호 매김)
  - 소요 기간
  - 비용
  - ⚠️ 주의사항 (WarningBox)

#### 섹션 3: 몽골에서 준비할 것
- 주몽골 한국대사관 위치 + 연락처
- 예치금 제도 설명 (몽골 고유 비자 예치금)
- 서류 공증 위치

#### 섹션 4: 비자 거절 시 대응
- 재신청 가능 시기 (6개월)
- 흔한 거절 사유 목록

#### 섹션 5: 불법체류 관련 경고
- WarningBox 사용
- 적발 시 절차, 출국 후 재입국 제한 기간
- 현실적 톤 (도덕적 설교 X, 팩트 중심)

#### 섹션 6: 유용한 링크
- LinkCard 컴포넌트 사용
- 주몽골 한국대사관, HiKorea, 몽골어 상담 가능한 곳

#### 섹션 2: 병원 이용법
- 의원(동네 병원) vs 대학병원 차이 설명
- 접수 방법 (StepList)
- 건강보험 유/무 시 비용 차이 (InfoTable)
- 약국 이용법 (처방전 시스템 설명)

#### 섹션 3: 통역 서비스
- 외국인 의료 상담 전화번호
- 다문화 지원센터 의료 통역

#### 섹션 4: 상황별 가이드

- 정신건강 → 다문화센터 상담 연결

#### 섹션 5: 주의사항
- WarningBox
- 보험 없이 응급실 이용 시 예상 비용
- 불법체류 상태에서도 병원 이용 가능 여부 (가능함)

### 5.5 송금/환율 가이드 (`/money`)

**제목:** Мөнгө шилжүүлэг, ханш (송금 & 환율)

**섹션 구성:**

#### 섹션 1: 인트로
- "한국에서 번 돈, 가장 이득되게 보내는 법"

#### 섹션 2: 송금 방법 비교
- **InfoTable 컴포넌트**

| 방법 | 수수료 | 소요시간 | 편의성 | MNT 직접 송금 |
|------|--------|----------|--------|---------------|
| 은행 해외송금 | 높음 | 1-3일 | 보통 | △ |
| 웨스턴유니온 | 보통 | 즉시~1일 | 높음 | ○ |
| GME 등 전문업체 | 낮음 | 1일 | 높음 | ○ |
| 환치기 (불법) | - | - | - | - |

#### 섹션 3: 각 방법 상세
- 은행 해외송금: 절차, 필요 서류
- 송금 전문 업체: GME, SBI, 모인 등 소개
  - 몽골 투그릭(MNT) 직접 송금 가능 여부
- ⚠️ 환치기 경고 (WarningBox)
  - 불법임을 명확히 안내
  - 사기 피해 사례
  - 적발 시 처벌

#### 섹션 4: 환율 팁
- TipBox
- 환율 확인 방법/앱 추천
- 소액 vs 대액 전략

#### 섹션 5: 한국 금융 상식
- ATM 수수료 절약법
- 체크카드 vs 신용카드
- 급여 수령 시 확인 사항 (세금, 4대보험)

#### 섹션 6: [Phase 2 예고]
- 실시간 환율 계산기 (KRW ↔ MNT) 예정

#### 섹션 3: 한국어 흔한 오해
- **UI: 아코디언 또는 카드 (아이콘 + 제목 + 펼치면 상세)**

1. **"네" ≠ "Yes"**
   - "네"는 "듣고 있다"는 신호일 수 있음. 동의가 아닐 수 있다.
   - 몽골인이 가장 많이 오해하는 것 중 하나.

2. **"괜찮아요"의 두 얼굴**
   - "괜찮아요" = OK일 수도, 거절일 수도
   - 손 흔들면서 "괜찮아요~" = "됐어요 (거절)"

3. **"밥 먹었어?" ≠ 진짜 밥 질문**
   - 한국식 안부 인사. "잘 지내?" 정도의 의미.

4. **반말 vs 존댓말 — 생존에 직결**
   - 직장 상사, 연장자에게 반말 쓰면 큰 문제
   - 몽골어에는 이 구분이 약하므로 특히 주의

5. **"형/오빠/언니/누나" — 한국 호칭 문화**
   - 이름 대신 관계 호칭 사용
   - 직장: "○○씨" 또는 직함

#### 섹션 4: 이것만은 조심! (문화 충돌 포인트)
- **UI: 아이콘 + 제목 + 상세 설명 카드**

🍻 **술 문화**
- 윗사람 앞에서 고개 돌려 마시기
- "한 잔만"은 한 잔이 아님
- 회식 문화 이해

🎂 **나이 & 서열**
- 한국은 나이 = 서열, 1살 차이도 중요
- 처음 만나면 나이 묻는 게 정상
- 몽골보다 훨씬 위계적

📱 **카카오톡 예절**
- 읽씹(읽고 안 답하기) = 무례
- 직장 상사 카톡에는 빨리 답하기
- 단체 채팅방 예절

🏠 **생활 매너**
- 층간소음 — 아파트에서 뛰지 않기 (매우 중요)
- 쓰레기 분리수거 — 안 하면 과태료
- 대중교통에서 전화 조용히

🤝 **직장 문화**
- 시간 엄수 (5분 전 도착이 기본)
- 상사보다 먼저 퇴근하기 어려운 문화
- 점심은 보통 같이 먹음

#### 섹션 5: 몽골인이 자주 하는 실수 TOP 5
- TipBox (또는 경험담 카드)
1. 직장에서 이름 불렀다가 혼남 → 호칭 문화
2. 쓰레기 아무데나 버렸다가 과태료 → 분리수거
3. "네" 해서 동의한 줄 알았는데 아니었음 → 의사소통
4. 밤에 시끄럽게 했다가 경비실 호출 → 층간소음
5. 회식에서 술 거절했다가 분위기 어색 → 술 문화

## 6. 공통 컴포넌트 정의

### 6.1 GuideCard
- 사용처: 메인 페이지
- 구성: 아이콘 (Lucide) + 제목 + 한줄 설명
- 동작: 클릭 → 해당 가이드 페이지 이동
- 스타일: white 배경, border, 호버 시 shadow + 골드 아이콘 전환

### 6.2 StepList
- 사용처: 비자 신청 절차, 은행 계좌 개설 등
- 구성: 번호 + 제목 + 설명 (접기/펼치기 가능)
- 스타일: 좌측 번호 원형(navy), 연결선

### 6.3 CheckList
- 사용처: 필요 서류, 도착 후 체크리스트
- 구성: 체크박스(인터랙티브) + 항목 + 부연설명
- 참고: 체크 상태 로컬 저장 (localStorage)으로 재방문 시 유지

### 6.4 WarningBox
- 사용처: 주의사항, 불법체류 경고 등
- 구성: ⚠️ 아이콘 + 제목 + 본문
- 스타일: terracotta 좌측 보더 + 연한 배경

### 6.5 TipBox
- 사용처: 꿀팁, 추천 정보
- 구성: 💡 아이콘 + 제목 + 본문
- 스타일: gold 좌측 보더 + 연한 배경

### 6.6 EmergencyBanner
- 사용처: 병원 페이지 상단, 메인 페이지
- 구성: 긴급 번호 3~4개 가로 나열
- 스타일: terracotta 배경 + 화이트 텍스트, 눈에 확 띄게
- 병원 페이지에서는 sticky 처리

### 6.7 InfoTable
- 사용처: 송금 비교, 비용 비교 등
- 구성: 반응형 테이블 (모바일에서는 카드 형태로 전환)

### 6.8 LinkCard
- 사용처: 외부 링크 모음
- 구성: 파비콘/아이콘 + 사이트명 + URL + 한줄 설명
- 동작: 클릭 → 새 탭 열기

## 7. SEO 전략

### 7.1 메타데이터
- 각 페이지별 title, description 몽골어로 작성
- Open Graph 태그 (SNS 공유 대비)
- 예시:
  ```
  title: "Солонгосын виз гарын авлага | KoreaMongol"
  description: "E-9, D-2, D-4 визний мэдээлэл, шаардлагатай бичиг баримт, өргөдөл гаргах дараалал"
  ```

### 7.2 sitemap.xml
- 자동 생성 (Next.js App Router의 sitemap.js 활용)

### 7.3 robots.txt
- 기본 허용, 관리자 영역 차단

### 7.4 구조화 데이터
- FAQ 스키마 (한국어 오해 섹션 등)
- HowTo 스키마 (비자 신청 절차 등)
